> [!summary] このページは「診断がつくまで」— **2 軸で走る**
>
> `#高齢者の転倒 s/o ●●による△△`（●● = 原因、△△ = 外傷）の形で立てる
>
> - **軸 1：転倒原因検索**（なぜ転んだか）— syncope / syncope mimic を外す
> - **軸 2：外傷評価とマネジメント**（どこを何で損傷したか）— ABCD → 頭から足まで
>
> 1. バイタル・意識・受傷機転で緊急度
> 2. **原因検索**：エコー / ECG / その他（起立・迷走・PE）
> 3. **外傷評価**：ABCD → 頭頸部 → 胸腹部 → 四肢
> 4. 頭部 CT の判断（抗凝固薬は閾値を下げる）
> 5. 帰宅 or 入院、**診断がついたら疾患ページへ** → [[頸椎損傷]] / [[大腿骨近位部骨折]] / [[脊椎圧迫骨折]] / [[脆弱性骨盤骨折]] / [[意識消失]]

## 最初に決めること

- **意識は清明か**
- **頭を打ったか / 打った可能性があるか**
- **抗凝固薬 / 抗血小板薬を内服しているか**
- **どうやって転んだか**（受傷機転）— 立位からの転倒か、階段か、失神が先行したか
- **外傷（出血・変形・疼痛）はあるか**
- **歩けていたか**（転倒後の歩行可否は骨折の除外にならないことに注意）

> [!danger] すぐ動く
>
> - **意識レベル低下（GCS ≤14）**
> - **瞳孔不同 / 新規の片麻痺 / けいれん**
> - **大量出血 / 止まらない出血**
> - **四肢の変形**（開放骨折・脱臼）
> - **頸部痛** → 頸椎保護
> - **ショック**
> - **失神が先行している** → 原因検索が必要
> - **大腿骨骨幹部の変形**（出血 1–1.5 L、直達牽引が必要）→ [[股関節痛]]

## 骨折診断の思考手順（骨折ハンター式）

> [!check] 整形外傷は「画像の前に考える」
>
> 内科：`身体所見 → 鑑別 → 検査`
> 整形外傷：`年齢 + 受傷機転 → 鑑別 → 骨折線イメージ → 画像で照合`
>
> 痛くて動かせない高齢者では、**身体所見を省略して画像へ直行**することも許容される（例：外旋・短縮位の大腿骨近位部骨折）。

### レントゲンで見える／見えない — 3 つの骨折タイプ

| タイプ | 特徴 | 対応の要点 |
|---|---|---|
| **①明確骨折** | 誰が見てもレントゲンでわかる | 見つけるだけ。調べれば対応できる |
| **②微妙骨折** | 見るべき所を知って初めて見える | **骨折線イメージ**を持ってから読影する |
| **③亡霊（occult）骨折** | レントゲンに絶対映らない | **鑑別に挙げた上で CT / MRI を追う** |

- **②③は鑑別を挙げられないと見逃す**。「レントゲンで異常なし」で思考停止しないこと。
- 骨折線は原則 **2 方向**（正面 + 側面、股関節は正面 + 軸位）で確認する。
- 骨皮質の途絶 / 2–3 スライスの連続 → 骨折線として扱う。

### マネジメントは 5 択で決める

| 分類 | 定義 | 典型例 |
|---|---|---|
| **即時整形** | 夜間・休日でも呼ぶ | 開放骨折、整復不能な脱臼、血管・神経損傷、脊椎破裂骨折、大腿骨骨幹部骨折 |
| **入院整形** | 入院 → 後日・翌朝コンサルト | 大腿骨近位部骨折（夜間）、脆弱性骨盤骨折、脊椎圧迫骨折で生活困難、occult の経過観察入院 |
| **帰宅手術** | 待機手術になるが帰宅可 | 転位の強い橈骨遠位端骨折、上腕骨近位部骨折 2 part 以上 |
| **帰宅保存** | 保存加療で外来フォロー | 転位の少ない橈骨遠位端骨折、安定型の上腕骨近位部骨折、鎖骨骨折、横突起骨折 |
| **再診不要** | 短期保存で済む | 打撲、scrape、軽度の指骨折 |

> [!tip] 上肢外傷は原則「帰宅」、下肢外傷は原則「入院」
> 上肢は歩けるので通院可、手術も待機的。例外は開放骨折 / 整復不能な脱臼 / 血管・神経合併の 3 つのみ。
> 下肢は荷重困難なら入院マネジメント。

### 高齢者の 4 大骨折（脆弱性骨折）— 頻度順

| 骨折 | 罹患率（女性／男性, /10000 人年） | 受傷機転 | 診断の要点 |
|---|---|---|---|
| **脊椎圧迫骨折** | 680 / 700 | しりもち・軽微 | 初回 Xp では診断不能、MRI で STIR 白・T1 黒 → [[脊椎圧迫骨折]] |
| **大腿骨近位部骨折**（頸部・転子部） | 50–76 / 33–36 | 転倒 | 外旋・短縮位で即 Xp。偽陰性 2–10% は CT / MRI → [[大腿骨近位部骨折]] |
| **橈骨遠位端骨折** | 75 / 19 | 手をついた転倒 | Xp で明確。sugar tong で固定 → [[橈骨遠位端骨折]] |
| **上腕骨近位部骨折** | 42 / 15 | 転倒・肩を打った | ほぼ明確骨折。三角巾 ± バストバンド → [[上腕骨近位部骨折]] |

- **原因不明の歩行困難で来院する高齢者の 4.4%** が受傷歴のない大腿骨近位部骨折（「なぜか急に歩けない」で受診）。**転んでいなくても骨折を鑑別に挙げる**。

## 問診

- **いつ、どこで、どうやって転んだか**
- **目撃者はいるか**（失神の有無の判断に重要）
- **転ぶ前に症状はあったか**：失神・前失神・めまい・胸痛・動悸・脱力
- **頭を打ったか / わからないか**
- **受傷後の健忘はあるか**
- **内服薬**：特に **抗凝固薬・抗血小板薬・降圧薬・睡眠薬・血糖降下薬**
- **既往歴**：心疾患・不整脈・てんかん・骨粗鬆症・認知症
- **骨折歴 / 手術歴**：大腿骨や人工関節の術後 → **インプラント周囲骨折**を鑑別に加える
- **ビスホスホネート内服歴**：脆弱性骨折のリピーターを見つけるきっかけ → [[骨粗鬆症治療]]
- **もとの ADL / 居住環境**：入院可否の判断に必須（要支援 1 か、車いすか、独居か）

## 身体診察（頭から足まで）

### 頭部

- **頭皮の腫脹・裂創・圧痛**（後頭部は必ず見る）
- **Battle 徴候**（乳様突起の皮下出血 → 側頭骨骨折）
- **Raccoon eyes**（眼窩周囲の皮下出血 → 前頭蓋底骨折）
- **耳・鼻からの液体漏出**（髄液漏）
- **瞳孔**（左右差・対光反射）

### 顔面

- 変形・腫脹・圧痛・開口障害・歯牙損傷

### 頸椎

- **正中圧痛** → 頸椎損傷の可能性 → **固定して画像評価**
- 高齢者では軽微な受傷でも起こりうる — 痛みがあれば安易に否定しない
- **画像を撮るか**：NEXUS / Canadian C-spine Rule（詳細 → [[頸椎損傷]]）
- **CT 後に MRI を追うか**は **①②③すべて満たせば MRI 不要** の判定 → [[頸椎損傷]]
    - ① **CT で骨折像なし**
    - ② **OPLL / OLF / DISH なし**
    - ③ **全身神経診察上、普段からの変化なし**
- ②の素地がある or 神経所見あり → **MRI を追う**（中心性頸髄損傷に注意）

### 胸部・腹部

- **胸郭の圧痛**（肋骨骨折）
- **呼吸音の左右差**
- **腹部の圧痛・膨満**（抗凝固薬内服中は腹腔内出血も考慮）

### 腰背部

- **椎体の棘突起叩打痛**：closed-fist percussion sign（CFPS）— 感度 87.5% / 特異度 90% 程度
- **傍脊柱筋の圧痛**（椎体から 1–2 cm 外側）→ 横突起骨折（高エネルギー）
- **背部全体の圧痛分布** → 圧迫骨折の責任椎体を推定
- 低エネルギー＋高齢者の腰痛 → 鑑別は **[[脊椎圧迫骨折]] の一択**
- 高エネルギー → **脊椎破裂骨折**（即時整形）、**横突起骨折**（帰宅 or 入院保存）

### 骨盤・股関節

- **骨盤の圧迫痛** → 骨盤骨折
- **鼠径部・大転子部の圧痛** → 大腿骨近位部骨折
- **下肢の短縮・外旋** → **大腿骨近位部骨折を強く示唆**（1 秒で判断できる）
- **股関節の内外旋で疼痛**（他動的に動かして誘発）→ occult の [[大腿骨近位部骨折]] を示唆
- **恥骨結合・坐骨結節・仙骨の圧痛**（臥位 / 股関節屈曲位 / 側臥位）→ [[脆弱性骨盤骨折]]
- 大腿骨近位部骨折術後の患肢痛 → **インプラント周囲骨折（ステム周囲骨折）**
- 股関節痛のフローは → [[股関節痛]]

### 四肢

- **変形・腫脹・圧痛・可動域制限**
- **手関節の変形（silver fork）** → [[橈骨遠位端骨折]]（手をついた場合）
- **肩を動かせない・変形** → [[上腕骨近位部骨折]] / 肩関節脱臼 / [[鎖骨骨折]]
- **足関節の腫脹** → [[足関節痛]]（Ottawa rule に準ずるが日本では診療補助ツール）
- **荷重できるか** → 歩行を試せる状態なら確認（ただし**歩行可＝骨折否定ではない**）

### 皮膚

- **擦過傷・皮下出血の分布** → 受傷機転の推定
- **頭部裂創** → 洗浄 + 縫合 or ステープラー
- **スキンテア** → 皮弁を戻して被覆
- **大転子部の新鮮皮下血腫** → 大腿骨近位部骨折の補助所見（感度 56%）

## 部位別 骨折の鑑別とマネジメント

> [!tip] 使い方
> 痛がっている部位から入る。**鑑別疾患**を挙げ、**骨折線イメージ**を持ってから画像へ進む。
> 年齢（高齢者 / 成人 / 小児）によって鑑別が大きく変わるため、先に年齢を意識する。

### 腰痛

| 鑑別 | 受傷機転 | 画像 | マネジメント |
|---|---|---|---|
| **脊椎圧迫骨折** | 低エネルギー（しりもち） | Xp では診断不能、MRI | 生活可能 = 帰宅保存 / 困難 = **入院整形** |
| **脊椎破裂骨折** | 高エネルギー（転落） | CT 必須 | 神経所見の有無によらず **即時整形** |
| **横突起骨折** | 高エネルギー or 直接打撲 | Xp 陰性、CT で確認 | **帰宅保存**、疼痛強ければ **入院整形** |

- 詳細 → [[脊椎圧迫骨折]]
- **楔形変形 = 新鮮骨折ではない**（新鮮骨折で楔形となるのは 10%、半年で 60%）。「潰れて見える椎体」は陳旧性の可能性が高い。
- **初回 Xp で責任病変は決められない**。MRI がなければ「圧迫骨折疑い」で ADL 基準で disposition。

### 股関節痛

| 鑑別 | 画像の要点 | マネジメント |
|---|---|---|
| **大腿骨近位部骨折**（頸部 / 転子部） | 正面 + 軸位。occult 2–10% | 日中 = 即時整形 / 夜間 = **入院整形**で翌朝コンサルト |
| **脆弱性骨盤骨折**（恥・坐・仙骨） | Xp 感度 46–50%、**CT** 追加 | 安定型 = 帰宅 or 入院、部分安定型 = **入院整形** |
| **筋挫傷**（内転筋） | MRI で STIR 白 | 保存加療。歩行困難なら入院 |
| **インプラント周囲骨折**（術後） | 術側の患肢痛。デバイス近傍 | **手術した病院**で再手術が原則 |

- 詳細 → [[股関節痛]] / [[大腿骨近位部骨折]] / [[脆弱性骨盤骨折]]
- **Xp 陰性時の 3 ステップ**：
    1. **股関節の内外旋**で疼痛が出るか → 大腿骨近位部骨折の occult
    2. **恥骨・坐骨・仙骨の圧痛** → 脆弱性骨盤骨折
    3. 両方陰性でも疼痛強ければ **MRI**（できなければ CT、それもだめなら入院観察）
- **歩けていても骨折は否定できない**。Occult の大腿骨近位部骨折で歩行できる例あり。

### 肩痛

| 年齢 | 多い鑑別 |
|---|---|
| **高齢者** | **上腕骨近位部骨折（大結節 / 外科頸）**、上腕骨脱臼骨折、鎖骨骨折 |
| **成人** | **肩関節脱臼**、鎖骨骨折、肩鎖関節脱臼、腱板損傷 |
| **小児** | 鎖骨骨折（一択に近い） |

- **上腕骨近位部骨折**は 80% が安定型 → 三角巾 ± バストバンドで帰宅保存。Neer 分類 2 part 以上は帰宅手術の可能性 → [[上腕骨近位部骨折]]
- **肩関節脱臼**は 90% 以上が前方脱臼。非整形外科医が整復すべき手技（外旋 + Milch / 肩甲骨回旋 + Stimson / 反対牽引）
- **鎖骨骨折**は「肩 2 方向」だけでは微妙骨折になるため、**鎖骨 2 方向**の追加撮影を忘れない → [[鎖骨骨折]]
- 合併する Bankart lesion / Hill-Sachs lesion は occult になり MRI で後日評価。帰宅時に説明しておく

### 手関節痛（手をついた FOOSH）

| 年齢 | 最多の鑑別 |
|---|---|
| **高齢者** | **橈骨遠位端骨折**（一択でよい、有病率が圧倒的） |
| **成人** | 橈骨遠位端骨折、**舟状骨骨折**（解剖学的嗅ぎたばこ窩の圧痛） |
| **小児** | 橈骨遠位端骨折（若木骨折）、**上腕骨顆上骨折**（肘） |

- **舟状骨骨折**は初回 Xp 陰性率が高い。嗅ぎたばこ窩圧痛があれば母指スパイカで固定 → 2 週後再検 or MRI
- 橈骨遠位端骨折の手術適応 3 つ：
    1. **活動性が高い**（紹介状に ADL を必ず書く）
    2. **関節内骨折**
    3. **整復して外固定してもずれる**
- 非整形外科医でも **sugar tong splint** は必須手技 → [[橈骨遠位端骨折]]

### 膝痛

| 鑑別 | 画像・所見 |
|---|---|
| **大腿骨遠位端骨折 / 脛骨高原骨折** | Xp + CT |
| **膝蓋骨骨折 / 脱臼** | Xp 側面で明瞭 |
| **ACL / MCL / 半月板損傷** | Xp 陰性で疑い → MRI（後日で可） |

- Xp の撮影判断：**Ottawa Knee Rule / Pittsburgh Rule**
- 高齢者の転倒で膝痛なら **大腿骨遠位端**や**脛骨高原**の occult を疑い、荷重困難なら CT 追加
- **TKA 術後** → インプラント周囲骨折

### 足関節痛

| 鑑別 | ポイント |
|---|---|
| **外果骨折（腓骨遠位端）** | 単独 / 三果骨折 |
| **脛骨遠位端 pilon 骨折** | 高所からの転落。CT 必須、即時整形考慮 |
| **Maisonneuve 骨折** | 足関節捻挫＋腓骨近位の圧痛 → **膝下まで触診** |
| **ATFL 損傷** | 腓骨遠位端前方の圧痛、腓骨先端の圧痛なし |
| **第 5 中足骨近位端骨折** | 足関節を捻って小指の付け根が痛い |

- **Ottawa Ankle Rule** は日本では診療補助ツール（Xp を減らすためより、撮る根拠づけに使う）

### 足部痛

| 鑑別 | ポイント |
|---|---|
| **踵骨骨折** | 高所からの転落 → **合併外傷の評価**（脊椎・骨盤） |
| **リスフラン関節脱臼骨折** | 中足骨基部に離解。見逃し多い |
| **中足骨骨折** | 直達外力 |
| **足趾骨折** | buddy taping で保存 |

## 頭部 CT の判断

> [!danger] 抗凝固薬内服中の頭部打撲は迷ったら撮る
> 抗凝固薬内服者は**軽症頭部外傷でも頭蓋内出血のリスクが高い**。
> GCS 15 でも、頭部打撲（疑い含む）があれば **CT を撮る閾値は低くする**。

### 撮るべき場面

- **GCS < 15**
- **頭部打撲あり**（疑いを含む）
- **抗凝固薬 / 抗血小板薬を内服中**
- **65 歳以上の頭部打撲**
- **受傷後の健忘**
- **嘔吐（2 回以上）**
- **新規の神経所見**
- **受傷機転が不明**（目撃者なし → 頭部打撲を否定できない）

### 撮らない選択肢がありうる条件（Falls Decision Rule）

以下の**すべて**を満たす場合：

- 頭を打った形跡がない
- 転倒前後の健忘がない
- 神経学的診察で新規異常がない
- 抗凝固薬を内服していない

（感度 98.6%、NPV 99.8%）

### 抗凝固薬内服者の注意

- 初回 CT で出血あり → **数時間後に再撮影を考慮**（急速増大のリスク）
- 初回 CT 正常でも → **遅発性出血に注意**（24–72 時間）
- **ワルファリン** → PT-INR 確認。出血あれば拮抗（ビタミン K ± PCC）
- **DOAC** → 施設の拮抗プロトコルに従う
- **出血あり** → **脳神経外科コンサルト**

## なぜ転んだか（原因検索）

**外傷の治療だけで終わらせない。** 特に高齢者では「なぜ転んだか」が重要。

> [!check] 原因検索の手順
>
> 1. **本人・目撃者の証言から失神の否定（−）** を確認
> 2. 否定できない → **syncope / syncope mimic の検索**（心腹部エコー + 心電図 + その他）
> 3. **何も見つからない** → **逆行性健忘の可能性**を考える（頭部打撲で記憶が飛び「つまずいただけ」になっている）

| 原因 | 確認すること |
|---|---|
| **つまずき / 滑り** | 環境要因。靴・段差・暗さ |
| **起立性低血圧** | 臥位→立位の血圧測定（≥20 mmHg 低下で陽性）。降圧薬・脱水 |
| **失神 / 前失神** | 目撃者の証言、前駆症状、ECG |
| **迷走神経反射** | 転倒時の状況（長時間立位、排尿後、食後、痛み刺激） |
| **不整脈** | ECG。徐脈・頻脈・pause |
| **低血糖** | 血糖測定。インスリン・SU 薬 |
| **てんかん** | 目撃者の証言、舌咬傷、尿失禁 |
| **薬剤性** | 睡眠薬・ベンゾ・降圧薬・オピオイド |
| **アルコール** | 飲酒歴、臭い、血中アルコール |

### 失神を疑うときの追加検索

**心電図で確認するもの**：

| 所見 | 疑う疾患 |
|---|---|
| VT / VF | 致死性不整脈 |
| AF（頻脈性） | 心原性塞栓・血行動態破綻 |
| III 度 AVB / Mobitz II 型 | 高度房室ブロック |
| SSS（洞不全症候群） | 徐脈性失神 |

**心エコーで確認するもの**（施行可能な場合）：

| 所見 | 疑う疾患 |
|---|---|
| low EF | 心不全による低灌流 |
| 壁運動異常 | ACS |
| HOCM（左室肥大 + SAM） | 肥大型心筋症による流出路閉塞 |
| 心嚢液 | 心タンポナーデ |
| 大動脈拡大 | 大動脈解離 |
| 腹部大動脈瘤 | AAA 破裂 |

**その他**：
- **PE** → 胸痛・SpO2 低下がないか
- **起立性低血圧** → 起立時血圧測定
- **迷走神経反射** → 転倒時の状況から判断

→ 何も見つからない場合 → **逆行性健忘** の可能性（頭部打撲で記憶が飛んでいるだけ）

> [!warning] 失神が先行している転倒
> 「転んだ」のではなく「倒れた」可能性がある。
> **目撃者がいない + 受傷機転が不明確** → 失神の可能性を考える。
> → **ECG は必ず取る**。心疾患の既往があれば循環器への相談も検討。
> — ESC 2018 失神ガイドライン

## 外傷評価 → マネジメント

### ABCD 評価（所見を具体に）

| 項目 | 拾いにいく疾患 | 所見 |
|---|---|---|
| **A**：気道閉塞 | 上気道閉塞 | **発声 / 口腔内異物 / 頸部腫脹 / stridor** |
| **B**：呼吸 | **フレイルチェスト** | 胸郭運動の**左右差**、胸部圧痛、奇異呼吸 |
| | **緊張性気胸** | **気管偏位**、頸静脈怒張、胸部聴診で呼吸音減弱、皮下気腫 |
| **C**：循環 | **FAST**（1 検査で 3 疾患） | **大量血胸 / 腹腔内出血 / 心タンポナーデ** |
| **D**：意識 | 頭蓋内病変 | GCS、瞳孔、四肢の動き（左右差） |

### 部位別の評価

| 部位 | 拾うもの |
|---|---|
| **頭部** | 頭皮腫脹・裂創・圧痛（**後頭部**）、Battle 徴候、Raccoon eyes、耳鼻液漏出、瞳孔（[[#頭部 CT の判断]]） |
| **頸部** | 正中圧痛 → 頸椎損傷（固定 + 画像）、**[[頸椎損傷]] の MRI 不要判定 ①②③** |
| **外傷性頸部症候群**（むち打ち様） | 頸部痛、可動域制限、後頭部痛、めまい、上肢の感覚異常。**高齢者は遅発性に症状が出ることあり**、帰宅時に再診基準を伝える |
| **胸部・腹部** | 胸郭圧痛、呼吸音左右差、腹膜刺激、**抗凝固薬内服中は腹腔内出血も** |
| **腰背部** | 棘突起 CFPS、傍脊柱筋圧痛（横突起）→ [[脊椎圧迫骨折]] |
| **骨盤・股関節** | 恥坐仙骨圧痛、股関節内外旋痛、大転子部血腫。[[股関節痛]] / [[大腿骨近位部骨折]] / [[脆弱性骨盤骨折]] |
| **四肢** | 変形・腫脹・圧痛・可動域制限・荷重可否。**大腿骨近位部骨折の短縮外旋**、橈骨遠位端（手をついた）、足関節 |

## 検査

必ずやるもの：
- **バイタルサイン**
- **画像**（受傷部位に応じて X 線 / CT）

状況に応じて追加：
- **血糖**（ふらつき・意識変容がある場合）
- **心電図**（失神・前失神・動悸が先行した場合、高齢者）
- **心エコー**（失神 + 心疾患の既往・リスクがある場合）
- **血液検査**（抗凝固薬内服中 → 凝固、高齢者 → 貧血・腎機能・電解質、骨折手術見込みなら**血型・クロスマッチ**）
- **起立試験**（起立性低血圧を疑う場合）
- **MRI**（occult 骨折を疑う時 — 大腿骨近位部 / 脊椎圧迫）
- **CT**（MRI が撮れない時の代用、高エネルギー外傷の脊椎評価、脆弱性骨盤骨折）

### 画像の優先順位（occult が疑わしいとき）

| 状況 | 第 1 選択 | 代替 |
|---|---|---|
| **大腿骨近位部骨折 occult** | MRI（感度・特異度ほぼ 100%） | CT（前額断 + 矢状断 + 冠状断で骨皮質の途絶を見る） |
| **脊椎圧迫骨折** | MRI（STIR 白・T1 黒） | **CT はあまり寄与しない**（レントゲンと情報量が大差ない） |
| **脆弱性骨盤骨折** | CT（単純で可、造影は血腫疑いのみ） | — |
| **舟状骨骨折 occult** | MRI | 2 週後の Xp 再検 |

## 疼痛管理

- **アセトアミノフェン**（アセリオ® 15 mg/kg 静注）— 高齢者の第一選択
- **NSAIDs は高齢者で避ける**（腎障害・消化管出血）
- **オピオイド**（フェンタニル・トラマドール）— アセトアミノフェン効果不十分時
- **大腿神経ブロック**（施行可能なら）— 大腿骨近位部骨折に有効（NICE 2023, AAOS）
- **血腫内麻酔（hematoma block）** — 橈骨遠位端骨折整復時
- **肩関節注射** — 肩関節脱臼整復時

> [!tip] 痛みが取れても disposition は変わりうる
> 脊椎圧迫骨折で VAS 10→3 になっても、**「この痛みで生活できるか」**で入院 / 帰宅を決める。
> 患者さん・介護者と一緒に判断するのが原則。

## disposition

### 帰宅できる条件

- 意識清明、神経所見なし
- 頭部 CT が正常（撮影した場合）、または CT 不要と判断
- 骨折がない or 保存的に帰宅管理できる骨折
- 歩行できる / 日常生活が自力でできる
- 転倒原因が明らかで重篤でない
- 抗凝固薬内服中でも CT 正常 + 経過良好なら帰宅可（遅発性出血の説明が必要）

### 帰宅時の説明

- **頭を打った場合** → **頭部外傷の注意事項**（嘔吐・意識変容・強い頭痛が出たら再受診）を必ず説明
- **骨折なしでも痛みが続く場合** → **再受診して再評価**（occult / 不全骨折の可能性）
- **独居高齢者** → 帰宅後の見守りが確保できるか確認
- **上肢外傷で固定した場合** → 指の屈伸を積極的に、挙上、しびれ・蒼白・強痛で即受診
- **肩関節脱臼整復後** → occult 骨折（Bankart / Hill-Sachs）の可能性を説明し整形フォロー

### 入院 or 専門科コンサルト

- **頭蓋内出血** → **脳神経外科**
- **大腿骨近位部骨折** → **整形外科**（48 時間以内、可能なら 24 時間以内の手術が目標）
- **大腿骨骨幹部骨折** → **即時整形**（直達牽引）
- **脊椎破裂骨折** → **即時整形**
- **頸椎損傷の疑い** → **整形外科 / 脊椎外科**
- **開放骨折 / 整復不能な脱臼 / 血管・神経損傷** → **即時整形**
- **脊椎圧迫骨折 / 脆弱性骨盤骨折で生活困難** → **入院整形**
- **失神の精査が必要** → 入院 or 循環器
- **歩行不能 / ADL が維持できない** → 入院
- **抗凝固薬内服中 + 頭部打撲** → 経過観察入院を検討

### 整形外科コンサルトの作法（骨折ハンター式）

> [!check] スムーズに通すコツ
>
> - **診断名は「大腿骨近位部骨折」ではなく「大腿骨頸部骨折」「大腿骨転子部骨折」**と区別して伝える
> - **Garden 分類**を添える（頸部骨折のみ。転子部は不要）
> - 橈骨遠位端骨折なら **活動性（ADL）と関節内 / 外**を紹介状に記載
> - 上腕骨近位部骨折なら **ADL + CT**（できれば）
> - Occult 疑いなら「**内外旋で疼痛あり、MRI で STIR 白・T1 黒**です」と言えると通る
> - 整形外科医が嫌がる時間帯（夜間・休日・週末）にあえてコンサルトしない手もある（overnight で翌朝に）

## 折りたたみ用メモ

### 見落としやすいもの

- **後頭部の打撲・裂創**（仰臥位のまま確認していない）
- **大腿骨近位部の occult 骨折**（初回 X 線で見えない → 痛みが続けば CT / MRI）
- **脆弱性骨盤骨折の仙骨部**（Xp 感度 7–44%、恥坐骨が折れていれば仙骨も疑う）
- **頸椎損傷**（高齢者の軽微外傷でも起こりうる、中心性頸髄損傷）
- **脊椎圧迫骨折**（背部痛として訴える、初回 Xp で責任椎体を決めない）
- **失神が原因の転倒**（「つまずいた」で片付けている）
- **抗凝固薬の確認漏れ**
- **ビスホスホネート未処方**（脆弱性骨折のリピーター予防）
- **インプラント周囲骨折**（術後患肢の転倒疼痛で必ず鑑別に）
- **舟状骨骨折**（嗅ぎたばこ窩圧痛。初回 Xp 陰性率が高い）
- **Maisonneuve 骨折**（足関節捻挫と思ったら膝下まで触る）
- **鎖骨骨折**（肩 2 方向だけだと写らない、鎖骨 2 方向を追加）
- **上腕骨脱臼骨折の外科頸骨折**（大結節は牽引で戻るが、外科頸は戻らない → 即時整形）
- **大腿骨骨幹部骨折**（出血 1–1.5 L、大腿神経ブロック + 直達牽引）

### やりがちなミス

- 「頭を打っていない」という本人の言葉だけで CT を省略する（目撃者がいなければ否定できない）
- 頭部だけ見て四肢・骨盤を診ない
- 外傷対応だけして転倒原因を考えない
- 高齢者の「歩ける」を過信して帰宅させる（翌日歩けなくなることがある、occult 骨折あり）
- 抗凝固薬内服者の頭部外傷で帰宅時説明が不十分
- **Xp で異常なし → 思考停止**（鑑別を先に挙げていないから）
- 脊椎圧迫骨折疑いで**楔形変形を見つけて満足**（陳旧性の可能性）
- **コンサルト時の診断名が「大腿骨近位部骨折」**（頸部 or 転子部で伝えないと整形が嫌がる）
- **NSAIDs を高齢者に出してしまう**（腎障害）
- **歩行 = 骨折否定**と考える（occult の大腿骨近位部骨折でも歩けることがある）

### 骨折ハンター的 TIPS（現場で効く）

> [!tip] 整形外傷 5 箇条
>
> 1. **画像より先に鑑別**を挙げる（年齢 + 受傷機転）
> 2. **骨折線イメージ**を持ってから読影する
> 3. 「**Xp 異常なし**」で止めない。occult を疑ったら CT / MRI
> 4. マネジメントは **5 択**（即時 / 入院 / 帰宅手術 / 帰宅保存 / 再診不要）で決める
> 5. コンサルトは **分類名 + 骨折線 + ADL** を添える

- **外旋 + 短縮位**を見たら身体所見を省略して Xp 直行でもよい（大腿骨近位部骨折）
- **恥骨・坐骨・仙骨は体表から触れる骨** → 圧痛があれば骨折と診断してよい
- **股関節・膝・肩のような関節部は触れないので他動運動で代用**（内外旋痛 = 大腿骨近位部）
- **Xp はほぼ必ず 2 方向**（正面 + 側面、股関節は正面 + 軸位）
- **CT で骨折線を見つけるコツ**：3 断面のうち 2 つで見える・2–3 スライス連続・骨皮質の途絶
- **MRI で新鮮骨折 = STIR 白・T1 黒**（筋挫傷も同じパターンで、場所が骨か筋肉かで区別）
- **高齢者転倒後の骨密度介入漏れ**：大腿骨近位部骨折・脊椎圧迫骨折の既往があって BP（ビスホスホネート）未処方なら介入機会

### 高齢者の転倒予防 — 再転倒を防ぐ視点（退院・帰宅時）

- 起立性低血圧の誘発薬（α 遮断薬・ニトロ）を減量
- ベンゾジアゼピン・睡眠薬の整理
- 白内障・視力の確認
- 住環境（段差・照明・浴室）
- 骨粗鬆症治療（BP 未処方ならビスホスホネート導入の提案）
