#骨折 #小児 #上腕骨顆上 #ER #神経血管障害

> [!summary] 上腕骨顆上骨折 = 小児肘外傷で最多、神経血管障害に注意
>
> - **4–8 歳の小児**、FOOSH で受傷
> - 小児肘関節骨折の **60%** を占める
> - **正中神経・前骨間神経・橈骨神経麻痺** の合併が多い
> - **Volkmann 拘縮**（上腕動脈損傷 → 虚血）は緊急対応
> - Gartland 分類で II 以上は **即時整形**

## 疫学

- 小児肘関節骨折の **60%** を占める（最多）
- 好発年齢 **4–8 歳**
- **伸展型 95%**（FOOSH）、屈曲型 5%（肘を直接打つ）

## 身体所見

- **肘の腫脹・変形**（骨折線の高さで段差）
- **肘窩のしわの消失**（腫脹で埋まる）
- **上腕遠位部の圧痛**
- **神経所見を必ずとる**（受傷直後の評価が重要）

### 神経所見（全例で実施）

| 神経 | 評価 | 麻痺時の所見 |
|---|---|---|
| 正中神経 | OK サイン・母指示指感覚 | OK サイン不能 |
| **前骨間神経** | **母指 IP 関節屈曲**（最多） | **丸を作れない（Pinch sign）** |
| 橈骨神経 | 手関節背屈 | 下垂手 |
| 尺骨神経 | 小指外転 | 鉤爪変形 |

### 血管所見

- **橈骨動脈触知**：触れなければ上腕動脈損傷を疑う
- **前腕コンパートメント緊満** → **Volkmann 拘縮の前駆症状**
- **5P**（pain, pallor, pulselessness, paresthesia, paralysis）

> [!danger] Volkmann 拘縮の 5P + 過度な疼痛
>
> - **受動伸展での激痛**はコンパートメント症候群を示唆
> - 上腕動脈損傷を放置すると **不可逆性の前腕虚血 → 拘縮**
> - **血管障害があれば即時整形 + 血管外科**

## 画像

- 肘 2 方向（正面・側面）
- **側面像で骨折線を探す**
- **Posterior fat pad sign** 陽性なら骨折あり
- **前方線（anterior humeral line）** が橈骨頭中央を通らなければ転位あり

## Gartland 分類

| Type | 所見 | 治療 |
|---|---|---|
| I | 転位なし | ギプス固定で保存 |
| **II** | 後方皮質のみ連続 | 整復 + ピンニング（**即時整形**） |
| **III** | 全周転位 | 整復 + ピンニング（**即時整形**） |
| IV | 多方向不安定 | 整復困難、即時整形 |

## マネジメント

### 非整形外科医の役割

1. **神経所見・血管所見の評価と記載**
2. **後方シーネ + 三角巾固定**（90° 屈曲位、過屈曲は禁忌 → 血流悪化）
3. **アイシング + 挙上**
4. **整形外科コンサルト**
   - Type I → 整形外来で保存
   - Type II–IV → **即時整形**
5. 血管障害・コンパートメント兆候があれば **緊急コンサルト**

### 小児への配慮

- 疼痛管理を十分に（アセトアミノフェン・イブプロフェン）
- **親への説明**：神経麻痺の多くは自然回復するが、3 ヶ月は経過観察が必要

## 帰宅時（保存例のみ）の注意

- **過度な疼痛・しびれ・指先の蒼白** → 即受診
- ギプス後 24–48 時間は腫脹ピークのため要注意
