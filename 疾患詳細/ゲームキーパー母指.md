#靱帯損傷 #母指 #UCL #スキー #ER

> [!summary] ゲームキーパー母指（スキーヤー母指）= 母指 MCP 関節の尺側側副靱帯損傷
>
> - **母指が外側に反って受傷**（スキーのストック・転倒）
> - **母指 MCP 関節尺側の圧痛 + 不安定性**
> - **Stener 病変**（断裂腱が母指内転筋腱膜で反転）があれば手術適応
> - 保存で済むか手術かは **不安定性**と **Stener 病変の有無**

## 受傷機転

- **スキーのストックを握ったまま転倒**（スキーヤー母指）
- **柵を握って倒れる**（ゲームキーパー：元々は狩猟者の慢性損傷に由来）
- **ボールキャッチ**（バスケ・バレー）
- **母指に外転・伸展方向のストレス**

## 解剖

- 母指 MCP 関節の尺側側副靱帯（**UCL**）が損傷
- UCL は母指の **つまみ動作**に必須
- 損傷放置 → 慢性不安定 → 関節症へ

## 身体所見

### 圧痛

- **母指 MCP 関節の尺側に限局した圧痛**
- 腫脹は軽度〜中等度

### 不安定性テスト

- 母指 MCP 関節を **30° 屈曲位**で UCL にストレスをかける
- **30° 以上の開大 or 健側より 15° 以上の差** → UCL 完全断裂を示唆
- 疼痛で評価困難なら局所麻酔下で再評価

## 画像

### レントゲン

- 母指 2 方向
- **基節骨基部尺側の撕脱骨片** を確認
- ストレス撮影で MCP 関節の開大を評価

### MRI / エコー

- UCL 断裂の評価
- **Stener 病変の検出**（手術適応の鍵）

## Stener 病変

- UCL が完全断裂し、**母指内転筋腱膜が断裂 UCL の近位に被さる**
- 靱帯が腱膜の外に反転 → **保存では絶対に治らない**
- **疑えば MRI / エコー**、証明されれば **帰宅手術**

## マネジメント

### 保存（部分断裂・不安定性軽度）

- **母指スパイカシーネ** 3–6 週
- その後リハビリ

### 手術（完全断裂・Stener 病変）

- **帰宅手術**（ORIF または 靱帯再建）
- 急性期に手術する方が予後良好（chronic では靱帯再建が必要）

## 非整形外科医の役割

1. **母指 MCP 関節尺側の圧痛**を疑う
2. **不安定性テスト**（疼痛で困難なら局麻下）
3. **レントゲンで撕脱骨片**確認
4. **母指スパイカシーネ固定 + 整形外科紹介**（1 週以内）

## 帰宅時の説明

- 「母指の安定性が失われている可能性があり、手術が必要なケースがあります」
- つまみ動作・握り動作の自粛
- 腫脹・疼痛強ければ再受診
