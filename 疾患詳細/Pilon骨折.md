#骨折 #Pilon #脛骨遠位端 #高エネルギー #ER #即時整形

> [!summary] Pilon 骨折 = 脛骨遠位端の関節内粉砕骨折、高エネルギーの象徴
>
> - **高所からの転落・交通事故**による軸圧で発生
> - **脛骨遠位端（距骨との関節面）に及ぶ粉砕骨折**
> - 軟部組織損傷が甚大 → 二期的手術が多い
> - **即時整形 / 入院整形**、外固定で一時安定化

## 疫学・受傷機転

- **高所からの転落**（屋根・梯子）
- **交通事故**（車の正面衝突でダッシュボードから踵への軸圧）
- 若年男性に多い

## 特徴

- 脛骨遠位端の **関節内粉砕骨折**
- **関節面（plafond）の陥没**
- 軟部組織損傷が甚大（open fracture 率高い）
- **両側・高位骨折**（脊椎・踵骨・膝）の合併を疑う

## 身体所見

- 足関節の **強い腫脹 + 変形**
- 血疱・皮膚水疱（軟部組織損傷の指標）
- **皮膚障害を伴う場合、ただちに手術はせず軟部組織の回復を待つ**（Staged treatment）

## 画像

### レントゲン

- 足関節 3 方向
- 脛骨遠位端の粉砕を確認
- **近位の腓骨骨折も忘れずに**

### CT

- **全例 CT**（関節面評価・手術計画）
- 骨片の立体的把握

### 全身検索

- 高エネルギー外傷 → **全身 CT**
- **踵骨・脊椎・膝の同時骨折**を疑う

## 分類（AO/OTA）

- **43 型**（脛骨遠位端）
- C 型（関節内）が Pilon 骨折

## マネジメント — 原則 **即時整形**

### ER での対応

1. **下腿シーネ固定**（膝関節含めて全下肢）
2. **挙上 + アイシング**
3. **CT + 整形外科コンサルト**
4. **開放骨折なら**：生食洗浄、抗菌薬、破傷風（詳細は [[開放骨折]]）
5. 合併外傷（踵骨・脊椎）を必ず検索

### Staged treatment（二期的治療）

- **Stage 1**：**即日外固定**（外固定器で仮固定）→ 軟部組織の回復を待つ（7–21 日）
- **Stage 2**：**軟部組織回復後に内固定**（プレート・スクリュー）

## 合併症

| 合併症 | 頻度・背景 |
|---|---|
| **軟部組織壊死** | 早期内固定で増加 |
| **創感染** | 粉砕・開放で高率 |
| **関節症**（外傷性） | 関節面の不整合で必発 |
| **コンパートメント症候群** | 下腿で必ず評価 |
| **深部静脈血栓症** | 長期 immobilization |

## 落とし穴

- Pilon 骨折を **足関節果部骨折**と同じ感覚で扱わない（手術タイミング・戦略が全く異なる）
- 軟部組織損傷が外見で軽く見えても、**水疱・血疱がある場合は必ず staged**
- 合併損傷（踵骨・脊椎）を見落とすと予後悪化

## 帰宅・入院判断

- ほぼ全例入院（外固定 + 観察）
- 多発外傷なら救命救急センター
