#骨折 #距骨 #無腐性壊死 #ER #高エネルギー

> [!summary] 距骨骨折 = 無腐性壊死のリスクが高く、即時整形の対象
>
> - **高所からの転落・交通事故**による軸圧 + 背屈強制
> - **距骨頸部骨折**が最多
> - **血流が逆行性**（遠位から近位へ） → **無腐性壊死率 20–100%**（転位度で変わる）
> - **即時整形**、転位があれば緊急手術

## 解剖と血流

- 距骨は足関節（上方）と距踵関節（下方）の間にあり、**関節軟骨で 60% 以上被覆**
- **表面の大部分が関節面 → 血管侵入部が少ない**
- **血流は主に後方・外側から逆行性に入る**
- 骨折で血管が途絶 → **無腐性壊死**（AVN）

## 受傷機転

- **高所からの転落**（垂直軸圧）
- **交通事故**（ダッシュボード損傷）
- **足関節背屈強制**

## 身体所見

- 足関節の強い **腫脹 + 変形**
- **距骨頸部（前面中央）の圧痛**
- 荷重不能
- 皮膚損傷（開放骨折）、神経血管障害

## 画像

### レントゲン

- 足関節 3 方向 + 足部
- **Canale view**（前後像を 15° 回外）で距骨頸部を評価

### CT（全例推奨）

- 骨折線の詳細、転位・粉砕の評価
- 手術計画

### MRI

- **AVN の評価**（受傷 3 ヶ月以降に実施）

## Hawkins 分類（距骨頸部骨折）

| Type | 所見 | AVN 率 |
|---|---|---|
| I | 転位なし | **0–15%** |
| II | 距踵関節脱臼 | **20–50%** |
| III | 距踵関節 + 足関節脱臼 | **50–90%** |
| IV | 距踵関節 + 足関節 + 距舟関節脱臼 | **ほぼ 100%** |

## マネジメント — 原則 **即時整形**

### ER での対応

1. **U-slab シーネ固定**
2. **挙上 + アイシング**
3. **CT + 整形外科コンサルト**
4. **転位骨折 → 緊急手術**（血流再開のため早期整復が AVN 予防）
5. 開放骨折なら洗浄・抗菌薬・破傷風

### 手術のタイミング

- **Hawkins II 以上は緊急整復**（6–8 時間以内が望ましいとされたが近年では軟部組織の状態優先）
- **Hawkins III–IV**：ただちに整復試行、整復できなければ観血的整復

## 合併症

| 合併症 | 頻度・背景 |
|---|---|
| **無腐性壊死（AVN）** | Hawkins 分類に比例 |
| **外傷性関節症** | ほぼ必発 |
| **皮膚壊死** | 脱臼の放置 |
| **感染** | 開放骨折 |

## Hawkins sign

- **受傷後 6–8 週**のレントゲンで、距骨天井部の **軟骨下骨透過性亢進**
- 陽性 = **血流が保たれている** → AVN リスク低い
- 陰性 = AVN のリスク

## 帰宅時の説明（保存例）

- **完全非荷重 2–3 ヶ月**
- 長期の follow（最低 2 年）で AVN 監視
- 腫脹・疼痛増強で再受診
