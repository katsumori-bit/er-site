#ER #救急 #骨折 #骨盤骨折 #脆弱性骨折 #高齢者 #occult

> [!summary] このページの使い方
>
> 1. 高齢者の転倒後**股関節痛**では [[大腿骨近位部骨折]] と同時に**必ず鑑別に**
> 2. Xp 感度は **46–50%** と低い → **CT で評価**
> 3. **恥骨・坐骨・仙骨の 3 点圧痛**を触診で探す（体表から触れる骨 = 圧痛で診断可能）
> 4. 大半は**保存加療**、痛みで生活困難なら**入院整形**

---

## 成人の骨盤骨折と違う

> [!warning] 混同厳禁
> 「骨盤骨折 = ショック・IVR」というイメージは**成人の骨盤骨折**。
> **高齢者の脆弱性骨盤骨折は全く別の病態**で、血行動態は安定、IVR も創外固定も不要。

| | 成人の骨盤骨折（一般的） | **脆弱性骨盤骨折** |
|---|---|---|
| 年齢 | 成人 | 高齢 |
| 受傷機転 | 高エネルギー外傷 | 転倒（しりもち） |
| 血行動態 | しばしば不安定 | **安定** |
| IVR | しばしば必要 | **不要** |
| 創外固定 | 時に必要 | **必要なし** |
| 対応 | 重症外傷センター | 一般救急で対応可能 |

---

## 疫学・主訴

- 転倒後の搬送時主訴：
    - **股関節痛・腰痛・殿部痛 94.4%**
    - 骨盤部痛そのものを訴えるのは **4.8%**
- つまり**「股関節痛」「腰痛」「殿部痛」を訴える高齢者では必ず鑑別に**

---

## 好発部位 — 恥骨・坐骨・仙骨の 3 点

- **恥骨骨折**
- **坐骨骨折**
- **仙骨骨折**（Xp 感度 7–44%、見逃し最多）

> [!check] どれか 1 つあれば他の箇所も探す
> 脆弱性骨折は多発しやすい。
> - **恥骨骨折 → 坐骨 / 仙骨も探す**
> - **坐骨骨折 → 恥骨 / 仙骨も探す**
> - Xp で恥坐骨骨折があれば **CT を追加して仙骨を確認**

---

## 身体所見 — 体表から触れる骨は圧痛で診断

| 骨 | 診察体位 |
|---|---|
| **恥骨** | 臥位、恥骨結合の直上を圧迫 |
| **坐骨** | **股関節屈曲位**で坐骨結節を圧迫 |
| **仙骨** | **側臥位**で仙骨後面を圧迫 |

- **体表から触れる骨 = 圧痛があれば骨折の可能性が高い**（骨盤骨折の身体診察で診断率 100% とする報告も）
- 関節内の骨（大腿骨近位部など）は直接触れないので、**内外旋痛**で代用

## 画像

### 第 1 選択：CT（単純で可）

- **Xp 感度 46–50%** — Xp だけでは信頼できない
- **単純 CT** で恥骨・坐骨・仙骨を評価
- 造影は不要（血行動態が安定していれば）
- **血腫疑い・血圧低下**があれば造影 CT

### MRI が必要な場面

- CT でも骨折線が不明瞭 / 仙骨不全骨折 occult の疑い
- CT で骨折あっても筋挫傷など鑑別が残るとき
- **STIR 白・T1 黒**で確定

## 分類と治療

### 非整形外科医目線のシンプル分類

| 分類 | 骨折部位 | 安定性 | 対応 |
|---|---|---|---|
| **安定型** | 恥坐骨のみ | 安定 | 荷重制限なし。痛みが弱ければ通院 or **帰宅保存** |
| **部分安定型** | 恥坐骨 + 仙骨 | 部分的不安定 | **4–8 週の荷重制限** → リハビリ目的で **入院整形** |
| 不安定型 | 完全転位・後方要素損傷 | 不安定 | 整形外科コンサルト（稀） |

- 大半は保存加療
- 通院可能な安定型でも**痛みで歩けないなら入院**
- コンサルト時は「恥骨・坐骨・仙骨のどこに骨折あり、安定型 or 部分安定型」と伝えるとスムーズ

### 疼痛管理

- **アセリオ®（アセトアミノフェン）15 mg/kg 静注**
- NSAIDs は高齢者で避ける
- 入院後はリハビリ（荷重制限の範囲で早期離床）

## Xp 陰性時のフロー

```
転倒後股関節痛 + Xp 陰性
 ├─ 股関節の内外旋で疼痛 → 大腿骨近位部骨折 occult を疑う → MRI / CT
 ├─ 恥骨 / 坐骨 / 仙骨の圧痛 → 脆弱性骨盤骨折を疑う → CT
 ├─ 両方陰性で疼痛強い → MRI で筋挫傷を含めて評価
 └─ すべて陰性 → 保存加療、再診指示
```

---

## 骨粗鬆症介入

- 脆弱性骨盤骨折は骨密度が低い場合に**ビスホスホネート適応**
- 整形外科フォローで骨密度測定 → 処方となることが多い
- 退院時に未処方なら整形外科外来での骨粗鬆症評価を依頼

---

## やりがちなミス

- 「Xp で骨折なし」で帰宅（Xp 感度 46–50% しかない）
- 恥坐骨骨折を見つけて満足（仙骨骨折が隠れている）
- 成人の骨盤骨折と混同して IVR を考える
- 身体所見で骨を触らない（恥骨・坐骨・仙骨は体表から触れる）
- NSAIDs を高齢者に処方
