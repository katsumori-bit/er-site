#ER #救急 #胸痛 #ACS #心筋梗塞 #肺塞栓 #大動脈解離 #気胸

> [!summary] このページの使い方
> 1. **5 大致死的胸痛** を 10 分以内に外しにいく（ACS / 大動脈解離 / 肺塞栓 / 緊張性気胸 / 食道破裂）
> 2. **来院 10 分以内に 12 誘導 ECG**、これを最優先で回す
> 3. ECG・問診・身体所見で **方向性** を決める
> 4. **採血（トロポニン・D-dimer）+ 胸部 X 線 / 必要に応じて CT** を出す
> 5. 該当する詳細ページに進む

> [!tip] 主訴が胸痛ではないとき
> - **呼吸困難が前景** → [[呼吸困難]]
> - **背部痛が前景** → 大動脈解離を疑い続けつつ整形外科的鑑別も考える
> - **動悸・失神が前景** → [[意識消失]]
> - **心窩部痛が前景** → [[腹痛]]（下壁梗塞・膵炎・胆道系も）

## 0. 来院 10 分以内にやること

> [!danger] 来院 → 10 分以内
> 1. **バイタル**（血圧は両上肢で測る）
> 2. **モニター装着**（SpO2・心電図・自動血圧）
> 3. **末梢ルート確保**（できれば 18–20G 2 本）
> 4. **12 誘導 ECG**（来院から 10 分以内 — AHA/ACC）
> 5. **採血**：トロポニン（hs-cTn）、D-dimer、CBC、生化、凝固、血型・クロス、血ガス
> 6. **酸素**：SpO2 < 90% または呼吸困難で投与（ルーチン高流量酸素は心筋障害を悪化させうる）
> 7. **アスピリン 162–325 mg 咀嚼**（ACS が否定できなければ早期に。出血リスクと禁忌を確認）

## 1. 致死的胸痛 5 つを外す

| 疾患 | 典型 | 初手 |
| --- | --- | --- |
| **ACS（STEMI/NSTEMI/UA）** | 圧迫感、冷汗、放散痛、嘔気、労作で誘発、休息で軽快 | **12 誘導 ECG + トロポニン**、STEMI なら直ちに循環器コール |
| **大動脈解離** | 突然の引き裂かれる胸背部痛、移動する痛み、左右上肢血圧差、新規 AR 雑音 | **造影 CT**（D-dimer 陰性なら除外に有用） |
| **肺塞栓（PE）** | 突然の呼吸困難 + 頻脈、胸膜痛、片側下肢腫脹、長期臥床・術後・OC | **D-dimer → 造影 CT（CTPA）**、Wells/PERC で前確率を絞る |
| **緊張性気胸** | 突然の片側胸痛 + 呼吸困難、片側呼吸音消失、頸静脈怒張、気管偏位 | **画像を待たず針減圧 → ドレナージ** |
| **食道破裂（Boerhaave）** | 強い嘔吐後の胸痛、皮下気腫、縦隔気腫、口臭 | **造影 CT or 食道造影**、外科コンサルト |

> [!danger] STEMI に出会ったら
> - **症状発症から 12 時間以内** → **再灌流療法**（PCI が第一選択、door-to-balloon ≤ 90 分）
> - **症状発症 12–24 時間** でも、症状持続・血行動態不安定なら PCI を考慮
> - 当院に PCI 設備がなければ **直ちに転送**（DIDO ≤ 30 分目標）
> - 詳細 → [[STEMI]] / [[循環器コンサルトの判断]]

## 2. 問診

### OPQRST で取る

- **O**nset：いつから、どんな状況で（労作中？安静時？嘔吐後？）
- **P**rovocation/Palliation：何で増悪・軽減するか（労作・体位・呼吸・食事）
- **Q**uality：圧迫感、絞扼感、引き裂かれる、刺すような、灼熱感
- **R**egion/Radiation：部位、放散先（左肩・顎・両腕・背部）
- **S**everity：強さ（NRS 0–10）
- **T**iming：持続時間、反復パターン

### 必ず聞くこと

- **発症様式**：突然 vs 徐々（突然なら大動脈解離・PE・気胸を強く疑う）
- **持続時間**：数秒（非心臓性が多い）vs 持続性（ACS・解離・PE）
- **随伴症状**：呼吸困難、冷汗、嘔気、失神、動悸、咳、血痰、発熱
- **危険因子**：年齢、性別、喫煙、高血圧、糖尿病、脂質異常、家族歴、避妊薬、悪性腫瘍、長時間臥床、手術、外傷、コカイン

> [!tip] ACS らしさを上げる所見
> - **放散痛（両腕 / 右肩 / 顎）**
> - **冷汗**
> - **嘔気・嘔吐**
> - **労作で誘発、休息で軽快**
> - **過去の ACS/PCI 歴**
>
> ACS らしさを下げる所見：
> - **体位変換で増悪**
> - **触診で再現できる胸壁痛**
> - **数秒の刺すような痛み**
> - **呼吸でのみ誘発される鋭い痛み**（ただし PE/胸膜炎は除外不要ではない）

## 3. 身体所見

- **バイタル**：両上肢の血圧（差 ≥ 20 mmHg → 解離示唆）
- **頸**：頸静脈怒張（PE / タンポナーデ / 緊張性気胸）
- **胸**：呼吸音の左右差・減弱（気胸・血胸）、皮下気腫（食道破裂・気胸）
- **心音**：新規雑音（特に AR → 解離）、S3/S4、心膜摩擦音
- **腹部**：圧痛（下壁梗塞は心窩部痛を呈することがある）
- **下肢**：片側腫脹（DVT）、左右差

> [!warning] 胸壁触診で再現できても安心しない
> 胸壁圧痛があっても、ACS / PE / 大動脈解離は否定できない（感度・特異度ともに不十分）。

## 4. 12 誘導 ECG の見方

| 所見 | 考える疾患 |
| --- | --- |
| **ST 上昇**（隣接 2 誘導以上） | STEMI → [[STEMI]] |
| **ST 下降 / T 波陰転化** | NSTEMI / UA、左主幹部・三枝病変、後壁梗塞 |
| **後壁梗塞パターン**（V1–V3 の R 増高 + ST 下降） | 後壁 STEMI → [[後壁梗塞]]（V7–V9 で再評価） |
| **de Winter T**（V1–V6 で上行 ST + 高 T） | LAD 近位部閉塞 → [[de Winterパターン]] |
| **Wellens 征候**（V2–V3 で T 波双相 / 深い陰転） | LAD 高度狭窄 → [[Wellens症候群]] |
| **新規左脚ブロック** | 当面は STEMI 等価として扱う（Sgarbossa 基準） |
| **右心負荷**（S1Q3T3、右軸偏位、V1–V3 T 陰転） | PE を示唆 |
| **電気的交互現象** | 心タンポナーデ |
| **広範な ST 上昇 + PR 下降** | 急性心膜炎 |

> [!check] ECG は 1 回で終わらせない
> - 症状あり・初回 ECG 非診断的 → **15–30 分後に再検**
> - 症状の変化、新たな所見の出現を捉える
> - 右側胸部誘導（V3R, V4R）→ 右室梗塞、後壁誘導（V7–V9）→ 後壁梗塞も忘れない

詳細は → [[入院時心電図]]

## 5. リスク評価

### HEART スコア（ED での ACS リスク層別化）

| 項目 | 0 点 | 1 点 | 2 点 |
| --- | --- | --- | --- |
| **H**istory | 非典型 | 中程度 | 典型的 |
| **E**CG | 正常 | 非特異的 | 有意な ST 偏位 |
| **A**ge | <45 | 45–64 | ≥65 |
| **R**isk factors | なし | 1–2 個 | ≥3 個 or 動脈硬化既往 |
| **T**roponin | 正常 | 1–3 倍 | >3 倍 |

- **0–3 点**：30 日 MACE < 2% → 早期帰宅検討（hs-cTn 0/3h プロトコル併用）
- **4–6 点**：観察入院 + 再検
- **≥7 点**：入院 + 早期侵襲的評価

### Wells / PERC（PE）

> [!check] PERC（全項目満たせば PE 検査不要）
> - 年齢 < 50
> - 心拍 < 100
> - SpO2 ≥ 95%
> - 片側下肢腫脹なし
> - 喀血なし
> - 4 週以内の手術 / 外傷なし
> - 既往の DVT / PE なし
> - エストロゲン使用なし

PERC 満たさない or 中等度以上 → **D-dimer**（年齢調整 cutoff: 50 歳以上は age × 10 ng/mL）→ 陽性なら **造影 CT（CTPA）**

## 6. 採血と画像

### 採血

| 項目 | 意義 |
| --- | --- |
| **hs-cTn**（高感度トロポニン） | ACS の中心的バイオマーカー。**0/1h or 0/3h プロトコル**で rule-out |
| **D-dimer** | PE / 解離の rule-out（陰性的中率を活用） |
| **BNP / NT-proBNP** | 心不全の合併評価 |
| **CBC** | 貧血、感染 |
| **生化（Cr）** | 造影 CT 前の腎機能 |
| **凝固（PT, APTT）** | 抗凝固薬の評価、tPA・PCI 準備 |
| **血型・クロス** | 解離・大量出血に備える |
| **血ガス（VBG/ABG）** | 呼吸不全、A-aDO2 |

### 画像

- **胸部 X 線**：気胸、縦隔拡大（解離）、肺水腫、肺炎、皮下気腫
- **造影 CT（CTPA / 大動脈 CT）**：PE、大動脈解離、肺梗塞
- **心エコー**：壁運動異常（ACS）、右心拡大（PE）、心嚢液（タンポナーデ）、AR（解離）
- **POCUS**：迅速ベッドサイド評価（特にショック例で有用）

## 7. ACS 疑いの初期治療（MONA-BASH の更新版）

> [!info] 「MONA」は古い
> 現在のガイドラインでは、ルーチンの酸素投与とモルヒネは推奨度が下がっている。

| 薬剤 | 用量・適応 |
| --- | --- |
| **アスピリン** | 162–325 mg 咀嚼。出血・アスピリン喘息は禁忌 |
| **P2Y12 阻害薬**（クロピドグレル / プラスグレル / チカグレロル） | PCI 経路で循環器と相談して選択 |
| **ヘパリン** | 体重あたり投与、APTT 1.5–2.5 倍 目標 |
| **硝酸薬（ニトログリセリン）** | 持続痛・高血圧で。**右室梗塞・PDE5 阻害薬使用中は禁忌** |
| **β 遮断薬** | 来院 24 時間以内に開始（ショック・心不全・徐脈・喘息は除外） |
| **スタチン** | 高用量で早期に開始 |
| **酸素** | SpO2 < 90% で投与。ルーチン高流量酸素は避ける |
| **モルヒネ** | 硝酸薬で改善しない強い痛みに限定（P2Y12 吸収遅延の懸念） |

## 8. 大動脈解離を疑ったとき

- **3 つのキー所見**：突然発症、移動する痛み、引き裂かれる性状（いずれも感度は完璧でない）
- **左右上肢血圧差 ≥ 20 mmHg**、新規 AR 雑音、神経脱落症状
- **D-dimer 陰性**（< 500 ng/mL or 年齢調整）+ **ADD-RS ≤ 1** → 解離の可能性は低い（IRAD 解析）
- 疑わしければ **造影 CT**（胸腹部）を躊躇しない
- 治療：**収縮期血圧 100–120 mmHg、心拍 60 以下**を目標に β 遮断薬（ランジオロール / エスモロール）を先行 → 必要なら血管拡張薬
- **Stanford A**（上行大動脈）→ **緊急手術**、**Stanford B**（下行）→ 内科治療基本

---

## 9. PE を疑ったとき

- **Wells / PERC** で前確率を絞る
- 中〜高リスク → **D-dimer → 陽性なら CTPA**
- 高リスク（ショック・血行動態不安定）→ **直接 CTPA or ベッドサイド心エコー**（右心拡大）
- 治療：抗凝固（ヘパリン → DOAC / ワルファリン）、massive PE は **線溶療法 / カテーテル治療 / 外科血栓摘除**
- 詳細 → [[肺塞栓]]

## 10. disposition

### 入院

- **STEMI / NSTEMI / 高リスク UA**
- **大動脈解離（A は手術、B は ICU 管理）**
- **PE（中等症以上は入院、低リスクは外来 DOAC も可）**
- **緊張性気胸 / 食道破裂**
- **HEART ≥ 4 / 持続痛 / 不安定**

### 帰宅検討

- **HEART 0–3 + hs-cTn 0/3h 陰性**
- **PERC 全項目陰性**
- **明らかな非心臓性原因が同定**（筋骨格痛、逆流性食道炎、肋間神経痛、不安発作）
- **再診基準を口頭 + 書面で説明**

> [!warning] 帰宅前に必ず伝える再診基準
> - 痛みが再発・悪化する
> - 呼吸困難が出る
> - 失神・冷汗・嘔気が出る
> - 痛みが背部に広がる
> - 片側下肢が腫れる
