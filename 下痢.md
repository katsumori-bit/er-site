#ER #救急 #下痢 #急性下痢 #脱水 #感染性腸炎

> [!summary] このページの使い方
>
> 1. **下痢が前景**ならこのページ
> 2. まず **「ただの胃腸炎にしてはいけない」下痢** を拾う
> 3. **炎症性下痢（血便・高熱・強い腹痛）か非炎症性下痢か**で枝分け
> 4. **補液（ORS 優先）→ 制吐 → 止痢薬／抗菌薬は適応を絞る**
> 5. 疾患の詳細はリンク先（[[急性胃腸炎]] / [[CDI]] / [[STEC・EHEC]] / [[旅行者下痢]] など）へ飛ぶ

> [!tip] 主訴が下痢ではないとき
> - **嘔吐が前景**で下痢は軽い → [[嘔気・嘔吐]]
> - **腹痛が前景** → [[腹痛]]
> - **血便・粘血便が主体** → [[血便・下血]]
> - **発熱が前景**で下痢は軽い → [[発熱]]

## 最初に決めること

- **不安定（ショック・敗血症・重度脱水）** ではないか
- **「胃腸炎」で済ませてはいけない疾患** ではないか
- **炎症性下痢（血便 / 38.5℃以上の発熱 / 強い腹痛）** か、**非炎症性下痢（水様・発熱なし）** か
- **抗菌薬使用歴・入院歴** があるか（→ [[CDI]]）
- **免疫不全・担癌・高齢・小児・妊婦** ではないか
- **渡航歴・集団発生・食歴** はあるか

> [!danger] 先に拾う赤旗
>
> - ショック / 重度脱水 / 起立性低血圧
> - 高熱（>38.5℃）+ 血便
> - **痛みの割に腹部所見が乏しい**（→ 腸間膜虚血）
> - 強い限局性腹痛・腹膜刺激（→ 外科腹症 → [[腹痛]]）
> - 嘔吐主体 + 排ガス排便停止 + 腹部膨満（→ 腸閉塞 → [[腹痛]]）
> - 糖尿病患者で嘔吐 + 下痢 + 多尿 + 過呼吸（→ [[DKA・HHS]]）
> - 高齢 + 心房細動 / 血管リスク + 腹痛（→ 腸間膜虚血）
> - 抗菌薬使用後 / 入院歴あり（→ [[CDI]]）
> - 担癌 / 化学療法中（→ [[好中球減少時発熱]]・腸炎・CDI）
> - 免疫不全（HIV、ステロイド、生物学的製剤）
> - 抗凝固薬内服 + 血便
> - 大量水様便で脱水進行が速い（→ コレラ様、Vibrio、毒素性）

## このページで最優先に外す疾患

下痢は急性胃腸炎が圧倒的多数だが、**胃腸炎で済ませると致命的**になる疾患がある。

- **腸間膜虚血**（高齢・Af・所見に乏しい激痛 + 下痢／血便）
- **絞扼性腸閉塞 / paradoxical diarrhea**（液状便だけが流れ出る）
- **[[DKA・HHS]]**（嘔吐 + 下痢 + 多尿 + Kussmaul）
- **[[副腎不全・副腎クリーゼ]]**（嘔吐 + 下痢 + 低血圧 + 低 Na・高 K）
- **敗血症**（感染巣不明の発熱 + 下痢）
- **重症 / 劇症 [[CDI]]**（高齢・抗菌薬曝露後、白血球著増、AKI）
- **[[STEC・EHEC]]**（血便 + 強い腹痛、抗菌薬と止痢薬は HUS リスク）
- **コレラ・Vibrio・毒素性**（米のとぎ汁様便、急速脱水）
- **異所性妊娠**（妊娠可能年齢で腹痛 + 嘔気は妊娠反応必須）
- **AKI / 低 Na / 低 K** による嘔気（電解質を測らずに帰さない）

> [!danger] 迷ったらまずこの 5 つを外す
> **腸間膜虚血・腸閉塞・DKA・敗血症・CDI**

## カルテで先に拾う

- **抗菌薬使用歴**（直近 8 週以内）→ **CDI**
- **入院歴 / 施設入所**（直近 12 週）→ **CDI**、ノロ集団感染
- **担癌・化学療法**（直近 2–3 週）→ 好中球減少、CDI、免疫関連腸炎（irAE）
- **糖尿病**（特に 1 型）→ DKA
- **副腎不全のリスク**（ステロイド長期内服中の中断、下垂体疾患）→ 副腎クリーゼ
- **抗凝固薬・抗血小板薬** → 血便、消化管出血
- **心房細動 / 血管リスク** → 腸間膜虚血
- **PPI 長期内服** → CDI、サルモネラ等のリスクが上がる
- **NSAIDs / 鉄剤 / メトホルミン / SGLT2 / 化学療法薬 / 新規開始薬**
- **妊娠可能性**
- **渡航歴・生水・生肉・生魚・買い置き食品・集団発生**

## 炎症性 vs 非炎症性で枝分け

| | 炎症性下痢 | 非炎症性下痢 |
|---|---|---|
| 便 | 血性・粘血便、少量頻回 | 水様、大量 |
| 発熱 | あり（しばしば >38.5℃） | なし〜微熱 |
| 腹痛 | 強い、しぶり腹（tenesmus） | 軽い〜中等度、疝痛 |
| 病原体 | サルモネラ、シゲラ、カンピロバクター、**STEC**、赤痢アメーバ、**CDI**、IBD | ノロ、ロタ、ETEC、Vibrio、毒素型食中毒、ウイルス性 |
| 検査 | 便培養 ± CDI ± STEC ± 採血 + 画像考慮 | 通常は不要、補液中心 |
| 抗菌薬 | 重症 / 高リスクで考慮（**STEC は避ける**） | 原則不要 |

詳細は → [[急性胃腸炎]]

## まず取るべき最小情報

- **発症様式 / 経過**：突然 / 徐々、何時間 / 何日
- **下痢**：回数、性状（水様・泥状・血性・粘血便）、量、米のとぎ汁様か
- **嘔吐の有無**（嘔吐前景なら → [[嘔気・嘔吐]]）
- **腹痛**：部位、強さ、持続 / 疝痛、移動の有無
- **発熱**の有無と程度
- **食歴**：生もの、貝類、鶏肉、卵、ハンバーグ、買い置き食品、集団発生
- **渡航歴**：地域、生水、ストリートフード
- **薬歴**：抗菌薬、PPI、化学療法、新規薬
- **入院歴 / 施設入所**：直近 12 週
- **既往**：糖尿病、IBD、HIV、担癌、ステロイド、副腎不全
- **女性**：最終月経、妊娠可能性

## 身体所見

- **バイタル + 起立性血圧変化**（起立で sBP 20 mmHg 以上低下、脈拍 30/分以上上昇は脱水を示唆）
- **意識・倦怠感**：DKA、副腎不全、敗血症のサイン
- **粘膜乾燥・皮膚ツルゴール・毛細血管再充満時間**
- **呼吸様式**：Kussmaul（DKA）
- **腹部**：膨満、腸雑音、限局圧痛、腹膜刺激、ヘルニア門
- **直腸診**：血便・タール便・宿便
- **CVA tenderness**：腎盂腎炎は嘔気・下痢のことも

> [!tip] 「胃腸炎にしては痛い」を見逃さない
> 急性胃腸炎の腹痛は**疝痛 + 排便で軽くなる**のが基本。
> **持続性・限局性・腹膜刺激・所見に乏しい激痛**があれば、外科腹症 / 虚血を考え直し → [[腹痛]]

## 検査の考え方

### 血液（基本セット）

- **CBC、CRP**
- **生化**：Na, K, Cl, BUN, Cr, Glu, AST, ALT, LDH, T-Bil, ALB, Ca, Mg
- **乳酸**（重症感・虚血疑い・ショック）
- **リパーゼ**（心窩部痛・持続嘔吐）
- **静脈血ガス**（重症脱水・代謝性アシドーシス疑い・DKA 疑い）
- **血糖**：必ず測る（DKA / HHS の見落とし回避）

### 尿

- **尿定性 + ケトン**（DKA・脱水評価）
- **妊娠反応**（妊娠可能年齢で原則全例）

### 便検査（適応を絞る）

ACG 2016 / IDSA 2017 では **急性下痢の全例に便培養は推奨しない**。下記を満たすときに提出する。

- **便培養（サルモネラ、シゲラ、カンピロバクター、STEC O157）**
    - 血便 / 粘血便
    - 高熱（>38.5℃）+ 強い腹痛
    - 重症脱水・敗血症像
    - 免疫不全・担癌
    - 14 日以上持続する下痢
    - 集団発生・食中毒疑い
- **STEC（O157）スクリーニング**：血便・小児・HUS 既往 → [[STEC・EHEC]]
- **CDI 検査**：詳細 → [[CDI]]
    - 直近 8 週の抗菌薬使用 / 入院 72 時間以降の下痢 / 高齢・施設入所・PPI / 担癌
    - **24 時間以内に 3 回以上の不形便**で検査適応
- **寄生虫検査**：渡航歴・MSM・慢性化・好酸球増多
- **ノロ PCR**：集団発生の疫学調査として

> [!tip] 便培養は「全例」ではない
> 健常成人の急性水様下痢の 90% 以上はウイルス性で自然軽快。
> 便培養を出すのは **炎症性下痢 / 重症 / 免疫不全 / 公衆衛生** の場面に絞る（ACG 2016, IDSA 2017）。

### 画像

| 場面 | 初手 |
| --- | --- |
| 腹膜刺激あり / 強い腹痛 | **造影 CT**（→ [[腹痛]]） |
| 嘔吐主体・膨満・排ガス停止 | **腹部 X 線 → 造影 CT** |
| 高齢 + Af + 所見に乏しい激痛 | **造影 CT（CTA）**（腸間膜虚血） |
| 重症 [[CDI]] 疑い（白血球著増・腹部膨満・低血圧） | **造影 CT**（toxic megacolon・穿孔評価） |
| 妊娠可能年齢の腹痛 | **骨盤超音波** |

## 重症度評価

> [!check] 重症化のサイン
>
> - sBP < 90、脈拍 > 120、起立性低血圧（+）
> - 意識変容
> - 経口摂取不能・尿量減少
> - AKI（Cr 上昇 / BUN/Cr 比 >20）
> - 著明な電解質異常（Na <130 or >150、K <3.0 or >5.5）
> - 著明な代謝性アシドーシス（HCO3 <15、乳酸 >2）
> - 高熱 >39℃ + 血便
> - 腹膜刺激・著明な腹部膨満
> - 妊婦、高齢、免疫不全、担癌、IBD 既往

## 治療

### 1. 補液（最優先）

ACG 2016 / WHO / WGO 2012 はいずれも **経口補液（ORS）が第一選択**。IV は経口困難・重度脱水のときに使う。

- **軽症〜中等症**：**経口補液（ORS）**
    - WHO 低浸透圧 ORS（Na 75 mEq/L、グルコース 75 mmol/L、浸透圧 245 mOsm/L）
    - 制吐後に少量頻回（5–10 mL を 5–10 分ごと）から開始
- **重症脱水・経口不能**：**細胞外液（ラクテック / 生食）IV**
    - 成人：**500–1000 mL ボーラス**を反応評価しながら
    - 小児：**20 mL/kg ボーラス**を 1–2 回
    - 心不全・腎不全・高齢者は過剰輸液を避けて **250 mL ずつ評価**
- **電解質補正**：低 K、低 Na、低 Mg、低 Ca、HCO3
- **DKA 疑い** → [[DKA・HHS]]

### 2. 制吐

- **オンダンセトロン 4 mg IV**（必要に応じて 8 mg まで、QT 延長注意）
- 詳細・選択肢の比較 → [[嘔気・嘔吐]]

### 3. 止痢薬（ロペラミド）

ACG 2016 / IDSA 2017 では **限定的に容認**。

> [!warning] ロペラミドを使ってよい場面・避ける場面
>
> **使ってよい**
> - 非炎症性（水様）下痢
> - 発熱なし
> - 血便なし
> - 旅行者下痢に抗菌薬と併用
>
> **避ける**
> - **血便 / 粘血便**
> - **高熱 + 全身症状**
> - **STEC（O157）疑い**（HUS リスク → [[STEC・EHEC]]）
> - **CDI 疑い**（toxic megacolon リスク → [[CDI]]）
> - **小児（特に乳幼児）**（2 歳未満は禁忌）
> - 重症潰瘍性大腸炎の活動期

- 用量：**ロペラミド 4 mg 初回 → 排便ごとに 2 mg、1 日 16 mg まで**

### 4. 抗菌薬（経験的治療）

ACG 2016 / IDSA 2017 / JAID/JSC 2023 ともに **健常成人の急性下痢に経験的抗菌薬を全例で使うことは推奨しない**。
理由：多くがウイルス性 → 効かない、耐性化、CDI 誘発、STEC では HUS リスク上昇。

> [!check] 経験的抗菌薬を考慮する場面
>
> - 重症（敗血症像、強い炎症性下痢、>6 回 / 日 + 発熱）
> - 免疫不全 / 担癌 / 高齢
> - 渡航者下痢（中等症以上）→ [[旅行者下痢]]
> - 院内・医療曝露の下痢で重症
> - **STEC（EHEC / O157）が疑わしい場合は使わない**

#### 経験的抗菌薬（成人・腎機能正常）

- **アジスロマイシン 1000 mg PO 単回**（または 500 mg/日 × 3 日）
    - 渡航者下痢（特に東南アジア・南アジア）で第一選択（カンピロバクターのキノロン耐性が高い）
    - 妊婦でも比較的安全
- **シプロフロキサシン 500 mg PO BID × 3–5 日**
    - 非渡航者の重症細菌性下痢
    - キノロン耐性、腱障害、QT 延長、CDI 誘発に注意
- **CTRX 1–2 g IV/日**
    - 経口不能、敗血症像、入院適応のとき
- **CDI が疑わしいとき** → [[CDI]]（フィダキソマイシン / バンコマイシン PO）

> [!warning] STEC（O157）疑いでの抗菌薬は避ける
> **抗菌薬と止痢薬は HUS のリスクを上げる**（小児で特に明確）。
> 血便 + 強い腹痛 + 発熱が乏しいパターンでは STEC を疑い、培養結果が出るまで抗菌薬は避ける → [[STEC・EHEC]]

### 5. その他

- **プロバイオティクス**：成人の急性下痢では一貫したエビデンスは弱い（ACG 2020 は積極推奨しない）
- **亜鉛**：途上国の小児下痢のみ WHO 推奨
- **食事**：早期に通常食へ戻すのが推奨（BRAT 食に固執しない）。乳製品は一時控えてもよい

## disposition

### 入院 / 観察を考える

- 重度脱水 / 経口摂取不能
- AKI、著明な電解質異常、代謝性アシドーシス
- 重症 [[CDI]] / 劇症 CDI
- [[STEC・EHEC]] + HUS 兆候（特に小児）
- 敗血症像
- 外科腹症 / 虚血が否定できない
- 免疫不全 / 担癌 / 妊婦 / 高齢で重症
- DKA / HHS / 副腎クリーゼ
- 診断不確実で再評価が必要

### 帰宅を考える

- バイタル安定、起立性変化なし
- 経口補液が可能（ED で 1–2 時間 ORS が入って嘔吐しない）
- 危険徴候・赤旗なし
- 重症化リスク（高齢・免疫不全・担癌・妊娠・DM）が低い
- 再診基準を具体的に説明できる

> [!warning] 帰宅時に必ず伝える
>
> - **再診基準**：高熱が続く、血便、強い腹痛、嘔吐で水分が入らない、尿量減少、ぐったりする、意識がぼんやりする
> - **経口補液の方法**：ORS を少量頻回（5–10 mL を 5–10 分ごとから）
> - **食事**：消化のよい炭水化物中心、無理に絶食しない
> - **手洗い・タオル分け**（家族内感染予防、ノロ・ロタ・CDI）
> - **就労・就学制限**：飲食業・医療・保育・学校の規定を確認

## 折りたたみ用メモ

### 外してはいけないもの

- 腸間膜虚血
- 絞扼性腸閉塞 / 腸閉塞
- [[DKA・HHS]]
- [[副腎不全・副腎クリーゼ]]
- 重症 [[CDI]] / 劇症 CDI
- [[STEC・EHEC]] + HUS
- 異所性妊娠
- 敗血症（菌血症 / 腹腔内感染）
- 急性虫垂炎・急性膵炎・胆管炎

### やりがちなミス

- **「胃腸炎」と決めつけて電解質と血糖を測らない**
- **若年女性で妊娠反応を取らない**
- **抗菌薬使用歴を聞かずに CDI を見落とす**
- **血便にロペラミドを処方する**
- **STEC 疑いで抗菌薬を出して HUS を増やす**
- **高齢 + Af + 「腹痛 + 下痢」を胃腸炎で帰す**（→ 腸間膜虚血）
- **嘔吐主体 + 排ガス停止を「胃腸炎」と判断**（→ 腸閉塞）
- **DKA をブドウ糖含有輸液で進行させる**

### 帰宅前に最低限

- バイタル + 起立性変化 + 体温の再評価
- 経口補液が ED で実際に入ったか
- 採血の結果（電解質・腎機能・血糖）を確認
- 再診基準を口頭 + 書面で伝える
- 制吐薬・ORS の処方とフォロー先
